          options: IGasEstimator | { config: IConfig },

      Should be instantiated using Config. Instantiating this class using GasEstimator represents the legacy version of this class. The legacy version contains methods like createREWATransfer, createDCDTTransfer, createDCDTNFTTransfer and createMultiDCDTNFTTransfer. This was done in order to minimize breaking changes in client code.
